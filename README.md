__/BS__ to bring up the options menu! or /bs opt, /bs options
* __Lock Frames__ Toggle Showing the Icon and Timer Frame and make them moveable 
* __Enable Screen Glow__ controls whether to add a screen glow when battle shout is missing AND in combat. Only shows in combat
* __Enable Icon__ will display an Icon when Battle Shout is missing. Hide it out of combat with the "Only in combat" box
* __Enable Timer__ will display a Timer when Battle Shout is up and a red ! when missing. Hide it out of combat with the "Only in combat" box
* __Early Warning Icon/Glow__ Icon and Glow will display __ Second before battle shout actually runs out, so you can refresh early
